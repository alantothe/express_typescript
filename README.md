# Node Express Server

A simple Express server built using TypeScript.

## Prerequisites

- Node.js (v14+)
- npm

## Installation

1. Clone the repository:
   - git clone 

2. Install the dependencies:
   - npm install

## Scripts

- **Development** 
   - npm run dev

- **Build** the project:
   - npm run build

- **Start** the server (after building):
   - npm start

## Server

By default, the server will run on http://localhost:3000.
